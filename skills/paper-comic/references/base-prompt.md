# 论文漫画生成基础规范

## 图片规格

- **类型**：漫画页面，多分格
- **方向**：竖版（Portrait）
- **比例**：2:3
- **分辨率**：高清，文字清晰可读

## 分格结构

### 分格边框
- 黑色边框线（1-2px）
- 分格间白色间隔（8-12px）
- 阅读顺序清晰（从左到右，从上到下）
- 分格大小有变化，形成视觉节奏

### 分格构图
- 每格有明确焦点
- 合理使用前景、中景、背景
- 镜头角度多样：平视、俯视、仰视、特写、全景
- 动作和视线引导阅读流向

## 文字元素

### 对话气泡
- 椭圆形，带指向说话者的尖角
- 白底黑边
- 手绘风格字体

### 旁白框
- 矩形框，位于分格边缘
- 浅黄色或米色底
- 用于解说、时间跳转、背景介绍

### 思考气泡
- 云朵形状，带小气泡指向思考者
- 用于内心独白

### 文字规范
- **所有文字必须是中文**
- 专业术语格式：中文（English）
- 字体清晰，大小适中
- 重点词汇可加粗

## 科学概念可视化

将抽象概念转化为具象画面：

| 概念类型 | 视觉表达 |
|---------|---------|
| 神经网络 | 发光节点和连线 |
| 数据流 | 光粒子沿路径流动 |
| 算法 | 几何图形、积木块 |
| 记忆/存储 | 书架、档案柜、图书馆 |
| 搜索/检索 | 放大镜、探照灯 |
| 学习/训练 | 攀登、成长的植物 |
| 创新/突破 | 光芒、破壳而出 |

**原则**：
- 用比喻代替公式
- 用图示代替文字描述
- 让抽象概念"看得见"

## 角色表现

### 导师角色
- 睿智、亲和的表情
- 解说时手势自然
- 可以指向图示或黑板

### 学生角色
- 好奇、专注的表情
- 提问时眼睛发亮
- 理解时恍然大悟

### 互动场景
- 对话自然，不生硬
- 有眼神交流
- 肢体语言丰富

## 语言要求

- 所有对话：中文
- 所有旁白：中文
- 专业术语：中文（English）
- 标签注释：中文
- 使用中文标点：""，。！？
